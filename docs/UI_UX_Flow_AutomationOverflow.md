### UI/UX Flow Documentation

#### Screen Inventory
1. Splash Screen.
2. Login Screen.
3. Create Automation (Naming).
4. Choose Trigger List A.
5. Select Action List A.
6. Choose Trigger List B.
7. Select Action List B.
8. Workflow Preview.
9. Execution History.
10. Profile & Settings.

#### Layout Descriptions
- **Splash:** Centered with large icon, title, subtitle, and CTA button.
- **Login:** Top-aligned form with back arrow and input fields.
- **Create Automation:** Form-based layout with name input and continue button.
- **List Screens:** Vertical lists with cards for triggers/actions.
- **Preview Screen:** Summary view with workflow steps and save button.
- **Execution History:** List of past runs with status indicators.
- **Profile Settings:** Profile header with menu items and logout button.

#### Navigation Map
1. Splash → Login/Signup → Automation Creation → Trigger Selection → Action Selection → Preview → Save.
2. Settings → Profile Edit → Logout → Login.
3. Main Dashboard → Execution History → Workflow Details.

#### Accessibility Guidelines
- High contrast for text and buttons.
- Touch targets meet minimum size requirements.
- Alt text provided for all icons and images.
